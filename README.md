# KinMap Prototype

一个面向村落亲属研究的人类学家谱原型工具，重点演示三类谱系组织方式在同一套界面中的可视化表达：

- 父系
- 母系
- 非单系 / 家屋制

这个原型适合用来讨论产品结构、田野记录方式和可视化交互，还没有接入正式后端。

## 当前内置项目

网站默认加载“六巷村家屋制项目”，数据来自 `六巷村土地承包公示表_1-55户.xlsx`：

- 55 个家屋制家户（由公示表承包方先行对应）
- 204 位登记家庭成员
- 186 块承包地
- 地块名称、编码、四至、面积与成员-户主关系

需要注意：这里的解释框架是家屋制，但这张表本身不是完整亲属访谈表。当前户间关系中的“同姓线索”“地块邻接线索”“姻亲姓氏线索”只是为了帮助田野核对，不能当作已经确认的亲属关系。

## 当前能力

- 村落尺度总览，适配约 50 余户家屋
- 可缩放、可拖动、可向四周扩展的画布
- 家屋节点可在画布中自由拖动
- 父系 / 母系节点可在各自画布中自由拖动
- 自定义画布区域，支持名称、颜色、位置与尺寸
- 家屋 / 谱系节点二维可视化
- 点击节点查看户详情与成员情况
- 点击关系线查看和编辑说明
- 选中家屋后高亮其直接关系
- 单户内部成员关系图
- 三种亲属组织模式切换
- 撤销 / 重做基础画布编辑
- 项目管理、信息录入、JSON 导入导出
- 本地保存与只读分享链接
- 真实田野表格数据的项目化录入展示
- 项目切换：可在六巷村家屋制项目与试验项目之间切换
- 档案桌 / 田野图纸风格的视觉系统，面向资料整理和关系判读

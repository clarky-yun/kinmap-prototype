# KinMap Prototype

一个面向村落亲属研究的人类学家谱原型工具，重点演示三类谱系组织方式在同一套界面中的可视化表达：

- 父系
- 母系
- 非单系 / 家屋制

这个原型适合用来讨论产品结构、田野记录方式和可视化交互，还没有接入正式后端。

## 当前能力

- 村落尺度总览，适配约 50 余户家屋
- 可缩放、可拖动、可向四周扩展的画布
- 家屋节点可在画布中自由拖动
- 父系 / 母系节点可在各自画布中自由拖动
- 自定义画布区域，支持名称、颜色、位置与尺寸
- 家屋 / 谱系节点二维可视化
- 点击节点查看户详情与成员情况
- 点击关系线查看和编辑说明
- 选中家屋后高亮其直接关系
- 单户内部成员关系图
- 三种亲属组织模式切换
- 撤销 / 重做基础画布编辑
- 项目管理、信息录入、JSON 导入导出
- 本地保存与只读分享链接
